2.Explanation

2.1 Description

This function is to help users change the ingredients in recipes. When a user lacks certain ingredients, he can choose the ingredients that can be replaced. If there is no chicken at home, then user can use this function to replace chicken with other ingredients.

2.2 User story and criteria

  • Substitution:
    User story Acceptance criteria
    As a user

    I want to substitute some ingredients in a recipe

    so that I can avoid to use some ingredients



    Criterion 0:
    Given I am a customer And I search a recipe And I am on recipe detail page

    When     I click on the "Substitution" button

    Then     Users can substitute the ingredients on the recipe with new ingredients And get new recipes.
