2.Explanation

2.1 Description

The function of this part is to help users replace the ingredients in the recipe. Because some users may be allergic to certain ingredients or other special circumstances, they can't use certain ingredients. Therefore, this function can help them replace inedible ingredients. This function can screen out the replaced menu for users to use.

2.2 User story and criteria

  • User story Acceptance criteria

    As a user,
    I want/can     to replace one ingredient with another
    so that     I can make a recipe which ingredients I lack are not included



    Criterion 0:
    Given I have logged in and entered the recipe details page

    When     I click on the substitution button under the search bar and input the recipe name in search bar and input "Ingredients" in substitution bar and touch the button search.

    Then     the system will give the user recipes which "Ingredients" have been replaced.
