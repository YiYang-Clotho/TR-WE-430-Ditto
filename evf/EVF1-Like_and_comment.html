2.Explanation

2.1 Description

In this function, the system allows users to click a heart icon to express affection for a recipe. If they love the post, they can click on the like button, the button will turn red. The system can sort recipes by “like” number. Users also are able to leave comments for the recipe and can view comments from other users with their avatars and created time.

2.2 User story and criteria

  • Like and comment:
    User story Acceptance criteria
    As a user,
    I want     to react to my favourite recipe and comment on it
    so that     I can provide my ideas to improve that recipe.



    Criterion 0:
    Given    I am on the main page and I choose a recipe, system switches to the recipe details
    When     I click the "like" button
    Then     “Like” button turns green, I have successfully pressed like
    When     I click the "like" button again
    Then     “Like” button turns blank, I have unlike the recipe

    Criterion 1:
    Given     I am on the main page and I choose a recipe, the system redirects to the recipe details
    When     I click the comment button (bubble chat symbol)
    Then     The system redircts to comment section AND I can post my comments
    When     I click "Return to main recipe
    Then     the system redircts to the main recipe page
