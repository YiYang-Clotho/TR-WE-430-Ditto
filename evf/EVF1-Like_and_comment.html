2.Explanation

2.1 Description

This is an extension to add to recipe details. Users can like and comment on posts they want to contribute or express personal opinion.

2.2 User story and criteria

  • Like and comment:
    User story Acceptance criteria
    As a user,
    I can/ I want     react to my favourite recipe and comment on it
    so that     I can provide my ideas to improve that recipe.



    Criterion 0:
    Given     I im in the main page and I choose a recipe, system switches to the recipe details
    When     I click the "like" button
    Then     “Like” button turns green, I have successfully pressed like
    When     I click the "like" button again
    Then     “Like” button turns blank, I have unlike the recipe

    Criterion 1:
    Given     I im in the main page and I choose a recipe, system switches to the recipe details
    When     I click the comment button (bubble chat symbol)
    Then     The system switch to comment section AND I can post my comments to public
    When     I click "Return to main recipe
    Then     System switches to the main recipe
