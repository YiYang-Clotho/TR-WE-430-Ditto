2.Explanation

2.1 Description

This is a function that is added to recipe details as an extended feature. In the bottom left corner of the recipe description you can find nutrition info and the blue text underlined: View info. When the user clicks on it, nutritional information such as fat, calories or protein is displayed as a chat bubble to simplify the UI. When users want to close nutrition info, they just need to press "Hide info" next to the bubble to hide that information.
