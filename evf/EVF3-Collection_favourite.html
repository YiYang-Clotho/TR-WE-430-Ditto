2.Explanation

2.1 Description

This function has been implemented in MVF. Users can click the heart button to collect recipes under the menu and users can quickly find the recipes they want on the profile page

2.2 User story and criteria

  • Collection and Favourite:
    User story Acceptance criteria
    As a user

    I want to collect a recipe

    so that I can find my favorite recipe



    Criterion 0:
    Given I have logged in And view the recipe page

    When     I see a picture of the recipe And at the bottom of each picture, there is a heart button. I click the button.

    Then     I can collect this recipe and add it to favorite list.
