2.Explanation

2.1 Description

Step-by-step instructions page should contain three parts which are recipe information, ingredients and instructions. For the first part, this section will show the user the recipe photo and its author. All the related tags will be shown as well. If users want to see more of similar recipes, they can click on the tag. The website will look through the database again and show the recipes containing this tag. The second part is ingredients, all the specific names and amounts will be shown. Besides, this part is designed to be a checkbox which users could check if they have those ingredients. For the last part is instructions, detailed steps will be offered with pictures. Users could use those for references.
