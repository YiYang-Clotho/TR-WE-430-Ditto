2.Explanation

2.1 Description

This is our one of key features. We allow registered users to have their online kitchen to record what the user has in the real kitchen like ingredients so that the system can recommend the suitable website by those ingredients.

2.2 User story and criteria

  • User story Acceptance criteria
    As a user

    I want/can search a kind of food ingredients in application

    so that give me the corresponding recipe to use this kind of food ingredients



    Criterion 0:
    Given user a navigation bar And a "Search" button

    When     user put the name of a food ingredient in the navigation bar And they touch research.

    Then     give them corresponding recipe And put other recommendation recipes at the bottom.
