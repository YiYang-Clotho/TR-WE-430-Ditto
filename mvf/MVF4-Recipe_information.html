Design diagram For recipe thumbnails:

Design diagram:

2.Explanation

2.1 Description

This feature allows users to select what recipe they need rapidly. We provide a small recipe card with the title, author and its tags. Registered users also can directly like the recipe through “like” button and save the recipe through a save button.

2.2 User story and criteria

  • User story Acceptance criteria

    As a user,
    I want/can     know about each recipe more graphically,
    so that     I could decide if I wanna cook this dish.



    Criterion 0:
    Given I have logged in and entered the main page

    When     I see the recipe picture I have interest in and click the picture

    Then     I entered a new page that has detailed information on the recipe including ingredients and cooking methods.
