2.1 Description

Each recipe page will have detailed information about ingredients and step-by-step instructions with pictures which will help user to cook.

2.2 User story and criterias

  • Step-by-step instructions:
    User story Acceptance criteria

    As     a user who is a starter in cooking
    I want/can     know the exact cooking steps of a recipe,
    so that     I could succeed more easily.



    Criterion 0:
    Given I have entered a specific recipe page.

    When     I decide I want to have a try on this recipe.

    Then    I scroll down and follow each step while cooking.
