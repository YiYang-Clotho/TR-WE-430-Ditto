2.Explanation

2.1 Description

The function of this part is to help users replace the ingredients in the recipe. Because some users may be allergic to certain ingredients or other special circumstances, they can't use certain ingredients. Therefore, this function can help them replace inedible ingredients. This function can screen out the replaced menu for users to use.
